Reuters Newswire Topics Classification

Sun 06 December 2020

The Reuters-21578 benchmark corpus is a publically available version of the well-known Reuters-21578 corpus for text categorization. It has been used in well known publications. The collection spans 10,788 documents from the Reuters financial newswire service, partitioned into a training set with 7769 documents and a test set with 3019 documents.

It is also available for download from UCI which includes a more extensive history of the data revisions.

I have used the dataset from keras.datasets for this project.

In this project, we will load the 'Reuters newswire topics classification' dataset and perform the following tasks:

  • Study the Reuters newswire dataset
  • Import the dataset from 'keras.datasets'
  • Create an MLP for classification of the dataset.
  • Print the test accuracy of the model.

Vectorizing sequence data...

Keras is a high-level neural networks API, written in Python and capable of running on top of TensorFlow, CNTK, or Theano. It was developed with a focus on enabling fast experimentation.

Tokenizer Class This class allows to vectorize a text corpus, by turning each text into either a sequence of integers (each integer being the index of a token in a dictionary) or into a vector where the coefficient for each token could be binary, based on word count, based on tf-idf.

tokenizer = Tokenizer(num_words=max_words)
